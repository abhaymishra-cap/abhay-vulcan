## Installation and start-up

* Pre-requisite: You should have `Node v14.21.3` installed
* Install dependencies using `npm`

```sh
npm install
```

## Scripts

* **install** - install the dependencies
* **clean install** - remove dependencies and reinstall the dependencies
* **start** - Serve in development environment on 8000 port
* **build** - Make production build
* **generate** - Used for generating containers/components
* **extract-intl** -  Extract I18n messages into JSON files for upload to locize or translation manager of your choice
